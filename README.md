# proyecto-AlgebraSuperior
El siguiente proyecto implementa una interfaz gráfica en python usando el modulo TKinter para resolver sistemas de ecuaciones lineales por método de gauss. 
El  sistema tiene los siguientes objetivos:
  - Ingresar un sistema de ecuaciones de forma gráfica.
  - Decir si el sistema dado tiene solución unica, en tal caso dar la solución.
  - Decir si el sistema dado no tiene solución.
  - Decir si el sistema dado tiene infinitas soluciones, en tal caso dar la solucion general de forma paramétrica.
